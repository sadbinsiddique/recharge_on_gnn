# SSGNN Formula

---

## Graph Construction Formulas

### Formula 1: Euclidean Distance Matrix

**Purpose:** Calculate pairwise distances between all facial landmark nodes

$$
D_{ij} = \sqrt{\sum_{k=1}^{d} (x_i^k - x_j^k)^2}
$$

**Where:**

- $D_{ij}$ = Euclidean distance between landmark $i$ and $j$
- $x_i^k$ = k-th coordinate of landmark $i$
- $d$ = Dimension of coordinate space (typically 2 for 2D landmarks)

**Example:** Distance between two facial landmarks at coordinates (0,1) and (1,0):

$$
D = \sqrt{(0-1)^2 + (1-0)^2} = \sqrt{2} \approx 1.414
$$

---

### Formula 2: Adjacency Matrix (K-Nearest Neighbors)

**Purpose:** Define graph connectivity by connecting each node to its K-nearest neighbors

$$
A_{ij} = \begin{cases} 1 & \text{if } j \in \text{KNN}_k(i) \\ 0 & \text{otherwise} \end{cases}
$$

**Where:**

- $A_{ij}$ = Adjacency matrix element
- $\text{KNN}_k(i)$ = Set of k-nearest neighbors of node $i$
- Typically $k = 8$ for facial landmarks

**Properties:**

- $A$ is symmetric: $A_{ij} = A_{ji}$
- Self-loops are excluded: $A_{ii} = 0$

---

### Formula 3: Degree Matrix

**Purpose:** Calculate the degree (number of connections) for each node

$$
D_{ii} = \sum_{j=1}^{n} A_{ij}
$$

$$
D = \text{diag}(D_{11}, D_{22}, ..., D_{nn})
$$

**Where:**

- $D$ = Degree matrix (diagonal matrix)
- $D_{ii}$ = Number of neighbors connected to node $i$
- $n$ = Total number of landmarks

**Example:** If a node has 8 connections, then $D_{ii} = 8$

---

### Formula 4: Laplacian Matrix

**Purpose:** Graph Laplacian capturing topological structure of facial landmarks

$$
L = D - A
$$

**Where:**

- $L$ = Laplacian matrix
- $D$ = Degree matrix
- $A$ = Adjacency matrix

**Properties:**

- $L$ is symmetric and positive semi-definite
- Essential for spectral domain analysis

---

## Spectral Domain Formulas (SPEGNN)

### Formula 5: Eigendecomposition

**Purpose:** Decompose Laplacian into eigenvalues and eigenvectors for spectral analysis

$$
L = V\Lambda V^T
$$

**Where:**

- $V$ = Matrix of eigenvectors (basis for Graph Fourier Transform)
- $\Lambda$ = Diagonal matrix of eigenvalues
- $\Lambda = \text{diag}(\lambda_1, \lambda_2, ..., \lambda_n)$

**Computational Note:** Use `scipy.linalg.eigh()` for numerical computation

---

### Formula 6: Graph Fourier Transform

**Purpose:** Transform spatial features into spectral domain using eigenvector basis

$$
\hat{x} = V^T x
$$

**Where:**

- $\hat{x}$ = Spectral representation of spatial signal $x$
- $V^T$ = Transpose of eigenvector matrix
- $x$ = Spatial domain feature vector

**Interpretation:** $\hat{x}$ represents signal components in frequency space defined by the graph structure

---

### Formula 7: Spectral Convolution

**Purpose:** Apply learnable spectral filters in the frequency domain

$$
y_{\text{spe}} = \sigma(V \Theta(\Lambda) \hat{x})
$$

**Where:**

- $\Theta(\Lambda)$ = Learnable spectral filter (function of eigenvalues)
- $\sigma$ = Non-linear activation function (ReLU)
- $y_{\text{spe}}$ = Spectral domain feature output

**Filter Design:** $\Theta(\Lambda) = \text{diag}(\theta_1, \theta_2, ..., \theta_n)$ where $\theta_i$ are learnable parameters

---

### Formula 8: Softmax Normalization (Spectral)

**Purpose:** Convert spectral features into probability distribution over emotion classes

$$
P_{\text{spe}}(c) = \frac{e^{z_c^{\text{spe}}}}{\sum_{j=1}^{C} e^{z_j^{\text{spe}}}}
$$

**Where:**

- $P_{\text{spe}}(c)$ = Probability of emotion class $c$ from spectral domain
- $z_c^{\text{spe}}$ = Raw logit for class $c$
- $C$ = Total number of emotion classes (typically 5-7)

**Output:** Probability vector $\mathbf{p}_{\text{spe}} \in [0,1]^C$ where $\sum_c P_{\text{spe}}(c) = 1$

---

## Spatial Domain Formulas (SPAGNN)

### Formula 9: Spatial Feature Aggregation

**Purpose:** Aggregate features from neighboring nodes using adjacency weights

$$
f_i^{\text{agg}} = \sum_{j \in \mathcal{N}(i)} A_{ij} \cdot w_j \cdot f_j
$$

**Where:**

- $f_i^{\text{agg}}$ = Aggregated feature for node $i$
- $\mathcal{N}(i)$ = Neighborhood of node $i$
- $w_j$ = Learnable weight for neighbor $j$
- $f_j$ = Feature vector of neighbor $j$

---

### Formula 10: Normalized Spatial Aggregation

**Purpose:** Normalize aggregation by node degree for fair contribution

$$
f_i^{\text{norm}} = \frac{1}{D_{ii}} \sum_{j \in \mathcal{N}(i)} A_{ij} \cdot f_j
$$

**Where:**

- $f_i^{\text{norm}}$ = Degree-normalized aggregated feature
- $D_{ii}$ = Degree of node $i$

**Effect:** Prevents high-degree nodes from dominating the aggregation

---

### Formula 11: ReLU Activation (Spatial)

**Purpose:** Apply non-linear activation to spatial aggregated features

$$
f_i^{\text{relu}} = \max(0, f_i^{\text{norm}} + b_i)
$$

**Where:**

- $f_i^{\text{relu}}$ = ReLU-activated feature
- $b_i$ = Bias term for node $i$

**Property:** Introduces non-linearity while preserving positive signal components

---

### Formula 12: Average Pooling (Spatial)

**Purpose:** Aggregate all node features into single representation capturing muscle synergy

$$
f_{\text{pool}} = \frac{1}{n} \sum_{i=1}^{n} f_i^{\text{relu}}
$$

**Where:**

- $f_{\text{pool}}$ = Pooled spatial domain feature
- $n$ = Total number of landmark nodes

**Interpretation:** Captures overall facial muscle activation pattern

---

### Formula 13: Softmax Normalization (Spatial)

**Purpose:** Convert pooled spatial features into probability distribution

$$
P_{\text{spa}}(c) = \frac{e^{z_c^{\text{spa}}}}{\sum_{j=1}^{C} e^{z_j^{\text{spa}}}}
$$

**Where:**

- $P_{\text{spa}}(c)$ = Probability of emotion class $c$ from spatial domain
- $z_c^{\text{spa}}$ = Raw logit for class $c$
- $C$ = Total number of emotion classes

**Output:** Probability vector $\mathbf{p}_{\text{spa}} \in [0,1]^C$

---

## Decision Fusion Formula

### Formula 14: Decision Fusion

**Purpose:** Combine spatial and spectral predictions for final emotion classification

$$
\hat{y} = \arg\max_c \left[ \alpha \cdot P_{\text{spa}}(c) + (1-\alpha) \cdot P_{\text{spe}}(c) \right]
$$

**Alternative (Max Voting):**

$$
\hat{y} = \arg\max \{P_{\text{spa}}, P_{\text{spe}}\}
$$

**Where:**

- $\hat{y}$ = Final predicted emotion class
- $\alpha$ = Fusion weight (typically 0.5 for equal importance)
- $P_{\text{spa}}$ = Spatial domain probability vector
- $P_{\text{spe}}$ = Spectral domain probability vector

**Fusion Strategies:**

1. **Weighted Average:** $\alpha P_{\text{spa}} + (1-\alpha) P_{\text{spe}}$
2. **Max Voting:** Choose highest across both domains
3. **Geometric Mean:** $\sqrt{P_{\text{spa}} \cdot P_{\text{spe}}}$
4. **Log-Sum Fusion:** $\log(P_{\text{spa}}) + \log(P_{\text{spe}})$

### Computational Complexity

- **Eigendecomposition (Formula 5):** $O(n^3)$ for $n$ landmarks
- **Matrix Multiplication (Formulas 6,7):** $O(n^2)$ per layer
- **Aggregation (Formulas 9,10):** $O(k \cdot n)$ where $k$ = neighbors per node

### Numerical Stability

- Use `scipy.linalg.eigh()` for Laplacian eigendecomposition (symmetric matrices)
- Add small regularization to Laplacian: $L' = L + \epsilon I$ if needed
- Apply logarithmic scaling for softmax stability (log-sum-exp trick)
