## 1.0 Pseudocode

```
Pipeline
%START%
     # Graph Construction
     > Calculate Euclidean distances between all landmark nodes.
     > Define Adjacency Matrix (A) by connecting each node to its K-nearest neighbors.
     > Define Degree Matrix (D) by summing the connections for each node.
     > Compute Laplacian Matrix: L = D - A

     # Parallel Feature Extraction
     ## Spectral Domain
     > Perform Eigen-decomposition on L to find basis V and eigenvalues Lambda.
     > Transform spatial data to spectral domain using Graph Fourier Transform (V^T * x).
     > Apply spectral convolution filters.
     > Compute probability y_spe using Softmax.

     ## Spatial Domain
     > Aggregate features from neighboring nodes using spatial convolution weights.
     > Apply ReLU activation and Average Pooling to capture muscle synergy.
     > Compute probability y_spa using Softmax.

     # Decision Fusion
     > model_result = argmax(y_spe, y_spa) 
%END%
```

## 2.0 Code Representation

```python
import numpy as np
from scipy.linalg import eigh

def compute_ssgnn_logic(landmarks, k_neighbors=8):
    """
    Implements basic logic of SSGNN base on Equations 1, 2, 3 and 4.
    """
    n = len(landmarks)
    
    # Graph Construction on Spatial Data
    # Calculate Euclidean distances between nodes
    dist_matrix = np.sqrt(np.sum((landmarks[:, np.newaxis] - landmarks[None, :])**2, axis=2))
    
    # Eq (2): Adjacency Matrix A (k-nearest neighbors)
    A = np.zeros((n, n))

    for i in range(n):

        # Find indices of k-closest neighbors (excluding self)
        nearest = np.argsort(dist_matrix[i])[1:k_neighbors+1]

        A[i, nearest] = 1
    
    # Eq (3): Degree Matrix D
    D = np.diag(np.sum(A, axis=1))
    
    # Eq (4): Laplacian Matrix L
    L = D - A
    
    # Spectral Domain Conversion 
    # Eigen-decomposition L = V * Lambda * V.T
    eigenvalues, V = eigh(L)  

    # V is the basis for Graph Fourier Transform
    Lambda = np.diag(eigenvalues)
    
    # Simulated Decision Fusion 
    # Mocking probabilities for demonstration of Eq (1)
    y_spe = np.array([0.1, 0.8, 0.1]) # Spectral output
    y_spa = np.array([0.2, 0.7, 0.1]) # Spatial output
    
    # Final Classification (Eq 1/15)
    final_emotion = np.argmax([y_spe, y_spa]) 
    
    return A, D, L, eigenvalues, V

pts = np.array([[0,1], [1,0], [0,0], [0,0]])
```

## 2.1 What is Eigenvectors (v) এবং Eigenvalues (λ)?

### Answer

In the context of the eigen-equation, the relationship between a matrix (A), an eigenvector (v), and an eigenvalue (λ) is written as:

$$Av = \lambda v$$

- **A** = Square matrix (linear transformation)
- **v** = Eigenvector (non-zero vector)
- **λ** = Eigenvalue (scalar)

## 3.0 Architecture Branches

### 3.1 What is SPAGNN?

## Explanation:

SPAGNN means Spatial Domain Graph Neural Network. It works on the face graph directly. Facial landmarks/pixels are nodes, and nearby nodes are connected using distance. It learns how facial points move together during expression.
## Pseudocode:
Input: Facial landmark graph
For each node:
    Find neighboring nodes
    Aggregate neighbor features
    Apply ReLU
    Apply average pooling
Apply fully connected layer
Apply softmax
Output: spatial prediction y_spa

## Code:
import torch
import torch.nn.functional as F

def spagnn_forward(X, A, W):
    # X = node features, A = adjacency matrix
    H = A @ X              # aggregate neighbor features
    H = H @ W              # spatial convolution weight
    H = F.relu(H)
    H = H.mean(dim=0)      # average pooling
    y_spa = F.softmax(H, dim=0)
    return y_spa
